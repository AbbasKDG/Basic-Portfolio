A lot more content is to be displayed here which would constitute the mass and bulk of text on this page, relating to myself and areas of interest to potential readership of this dummy page. This page, while built on the mentality of a wireframe, was fleshed out and finally and successfuly implemented as the recommeneded assignment. After a number of iterations the need for frameworks such as bootstrap becomes apparent to the designer. The need to create a new template with its unique nuances and quirks and having to fine tune these for every occassion while educational soon devolves into a frustrating exercise.

There were a number of things I realized after completing this assignment. Lets start with the fact that there is no alternative to pencil and paper when it comes to creating the blueprint of the wbsite. It was vital to visualize the pages as creations comprising containers especially if one style sheet was to be used. However the realization that another style sheet can be used is also a useful tool that I will aim to keep in mind for my next assignment. Content is the first priority of a website designer. It is only when the content is at hand that one can begin to go about experimenting with the CSS to arrange it into its mosaic. And mosaic like it is. Juggling sections, classes and ids it is easy to get lost in the colorful maze of css functionality and the degree of interaction it creates on the webpage. Theres more to be said ofcourse but maybe on another page at another time. For now this version of the project has hit its end date as a basic protfolio webpage.
